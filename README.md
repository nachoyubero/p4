# La Buena Mesa - Restaurante

Página web estática para un restaurante ficticio, desarrollada con HTML y CSS puro.

## Páginas

- **index.html** — Página principal con información del restaurante, historia, vídeo y equipo de chefs.
- **carta.html** — Espacios del restaurante y menús degustación disponibles.
- **carrito.html** — Resumen del pedido con tabla de precios.
- **contacto.html** — Formulario de reserva.

## Tecnologías

- HTML5
- CSS3
